Instructions

  1. Start bridge server in your Codespace terminal: cd esp32-codespace-debug-bridge/bridge-server && npm start
  2. Forward port 8081 in Codespaces UI
  3. Connect ESP32 via USB to your local machine
  4. Click "Connect Serial & WebSocket" above
  5. Start GDB in Codespace terminal: 
    xtensa-esp32-elf-gdb build/distance.elf
(gdb) target remote localhost:3333
