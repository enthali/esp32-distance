üìñ Troubleshooting

Port not appearing?

  • Check USB cable connection
  • Try a different USB port
  • Verify ESP32 drivers are installed on your host machine
  • Close other applications using the serial port (Arduino IDE, PuTTY, etc.)

Flash failed?

  • Hold BOOT button on ESP32 while connecting
  • Try pressing RESET button before flashing
  • Verify firmware was built successfully
